# bidsniffer
BidSniffer is an AI-powered construction opportunity intelligence platform that continuously collects public bid opportunities, classifies them by trade and region, and helps contractors identify projects worth pursuing.

## Features

- Automated bid harvesting
- Trade-specific categorization
- AI-generated project summaries
- Contractor profile matching
- Public opportunity search engine
- Historical bid database
- SEO-optimized opportunity pages
- Daily opportunity updates

## Initial Coverage

- Colorado
- Electrical
- HVAC
- Roofing
- Concrete

## Long-Term Vision

Become the most useful public intelligence platform for construction opportunities by helping contractors spend less time searching and more time bidding.
